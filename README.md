# بوت ذكرني (Line Bot)

بوت لنشر الأذكار والأدعية والأحاديث القصيرة عبر LINE.

## التشغيل

1. عدل ملف `.env` وضع فيه:
   - LINE_CHANNEL_ACCESS_TOKEN
   - LINE_CHANNEL_SECRET

2. ثبت المكتبات:
   ```bash
   pip install -r requirements.txt
   ```

3. شغل البوت محلياً:
   ```bash
   python app.py
   ```

4. ارفعه على Render/GitHub.
